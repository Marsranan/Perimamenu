# Perima Weekmenu

Vercel-klaar Next.js project voor het beheren en delen van een weekmenu.

## Lokaal starten
```bash
npm install
npm run dev
```

## Deploy naar Vercel
1. Upload deze map naar GitHub
2. Maak een nieuw project in Vercel
3. Import de repository
4. Klik op Deploy

## Wat deze MVP doet
- Gerechten beheren
- Weeknummer en jaar kiezen
- Automatische datumrange per week
- Delen via kopiëren of native share
- Opslag in localStorage
